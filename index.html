Requirements

!doctype html

Opens the page, makes the browser aware of the document it is dealing with.

html

Open and close the document with this tag.

html lang=en

Language declaration for search engines.

div

Section out your webpage, non semantic.

head

Where the tab information goes, like the title and metadata.

title

Goes within the head tag as the title for the tab.

H1-H6

Header tag, H1 being the largest and H6 being the smallest.

p

Just a paragraph tag, use for standard text.

strong

Makes a section of text look bold.

em

Makes a text look like italics.

span style="text-decoration: underline;"

Underline a section of text.

!--- ---

Comment, not visible on live page.

hr /

Horizontal line.

br /

Line break.

q

Quotation.

blockquote cite=""

Quotation with citation.

abbr title=""

Marking abbreviation for search engines.

a href=""

Anchor element, basic hyperlink.

a href=mailto:""

Anchor for sending emails.

a target=""

Specifies where to open the link (_blank, _self, _parent, _top, _framename).

meta charset="" /

Tells browset what character set to use.

Semantic HTML

header

Marks the heading of a web page.

nav

Marks page navigation.

main

Marks the main section of text on a web page.

article

Mark a section of text and media that can operate in isolation.

section

Mark headers of different sections of the web page.

aside

Mark a section of text that is not crucial to the main bulk of text.

footer

Mark the footer of a web page.

address

Contact information.

Media

img src""

An image.

img alt=""

Description of image if it doesn't load or for the visually impaired.

img height="" width=""

Alters image dimensions.

video

Header for video media.

video controls

Adds controls for videos.

video width="" height=""

Alters video dimensions.

source src=""

Picks image source.

source type="video/"

Decides video format.

video""/video

Text to show if video doesn't load.

audio

Audio file.

source src=""

Picks file source.

source type="audio/"

Picks audio file type.

audio controls

Generates controls for audio player.

audio""audio

Message for when audio isn't playable.

iframe

Inline frame. Used to link and display other webpage documents within the HTML.

iframe src=""

Defines the source of the inline frame.

iframe width="" height=""

Defines iframe size.

iframe name=""

Defines the name of the iframe.

iframe sandbox

Applies restrictions to the iframe.

ol

dl

dt

Title of definition.

dd

li

List item.

Tables

table

Open and close a table.

caption

Used to caption the head of the table

tr

Used to create a table row.

td

Table data.

thead

Used to create table headers.

th

Used instead of td to mark table headers.

tbody

Used to create the table body.

tfoot

Used to create the table footer.

th/td colspan=""

Used to make heders and data span more than one column.

col

Used to mark out columns within a colgroup element for styling.

col span=""

Mark out styling span.

col style=""

Decide column style.

colgroup

Mark out groupa of columns for styling.

Forms

form

Mark out the start and end of a form.

form action=""

Marks out where to send the form data once completed.

form method="get/post"

Specifies the HTTP method to use when sending form-data.

form name=""

Specifies the name of the form.

autocomplete="yes/no"

Specifies whether or not a form will autocomlete.

form min="" max=""

Sets limit for number based forms.

form step=""

Number step for number based forms.

form value=""

Name for a specific value.

form type=""

Specifies form type.

option

Options for multiple choice forms.

textarea

Text area for additional input and prewritten text.

textarea cols="" rows=""

defines amount of columns and rows for text area.

select

Created options for dropdown list in conjunction with 'option'.

optgroup

Specifies groups for options.

fieldset

Group related form elements under one parent.

datalist

List of pre-defined options for an 'input' element.
